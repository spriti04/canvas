# :art: Drawing App(Canvas)
This is a drwaing platform where user can draw and erase on a canvas.

## Features
 - :pencil2:  Free-hand drawing
 - :art: Color selection
 - :boom: Brush size adjustment
 - :wastebasket: Clear canvas

## Technologies Used
 - HTML
 - CSS
 - JavaScript

## How to Run
Open `index.html` in your browser
